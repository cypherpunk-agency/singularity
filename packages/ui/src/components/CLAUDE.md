# Component Guidelines

## PROTECTED (do not modify without permission)
- Chat.tsx - Primary user interaction
- Layout.tsx - App shell and navigation
- Status.tsx - Agent status display
- AppProvider.tsx - Context providers

## MODIFIABLE (agent can improve autonomously)
- Files.tsx - File browser
- Outputs.tsx - Agent outputs
- History.tsx - Run history
- Jobs.tsx - Job tracker
- FileViewer.tsx - File content viewer

## New Components
Ask permission before creating new components.
